Vehicle Categories

We offer training for the following categories of vehicles:-

Class A1(Moped)
Class A2(Light motorcycle)
Class A3(Motorcycle taxis couriers-three wheelers e.g. tuktuk)
Class B1(Light vehicle)
Class C1(Light Trucks)
Class D1(Van)
Class D2(Mini-bus)
Class D3(Large bus)

A1- Enables one to ride a motorcycle
A2- Enables one to ride a p.s.v motorcycle
A3- Enables one to ride a p.s.v three-wheeler motorcycle
B1- Enables one to drive a light passenger car
One can drive either manual or automatic gearbox(can carry upto 7 passengers
C1- Enables one to drive a light truck equipped with either manual or automatic gearbox.
D1- Enables one to drive a van with maximum of 14 passengers(matatu)
either one with manual or automatic gearbox
D2- Enables one to drive a bus with 14 to 32 passengers(either manual or automatic)
D3- Enables one to drive a bus with 33 or more passengers(either manual or automatic)
